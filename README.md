# Vancouver Hockey

Vancouver Hockey is a fansite about several ice hockey teams based in Vancouver, British Colombia, Canada. It includes team descriptions, history, roster and photos/videos.
It is a static website built with HTML and CSS only. It's a project for Uvod u Web course at University of Zadar.

## Pages

- `index.html` - Home page with introduction to the project theme
- `pages/canucks.html` - Page about the Vancouver Canucks
- `pages/goldeneyes.html` - Page about the Vancouver Goldeneyes
- `pages/abbotsford.html` - Page about the Abbotsford Canucks
- `pages/contact.html` - Contact page with an HTML form

## Features

- Multi-page static website
- Responsive layout for mobile and desktop
- Mobile hamburger navigation without JavaScript
- Contact form with labels and HTML validation
- CSS variables for colors, spacing, and reusable styling
- Basic accessibility features:
  - skip link
  - visible focus states
  - `alt` text for images
- Basic SEO:
  - page titles
  - meta descriptions
  - Open Graph tags
